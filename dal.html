T20 Cricket Analytics Dashboard

Tools:

Power BI, Python, Pandas, Jupyter Notebook


This project focuses on analyzing T20 cricket match data to provide actionable insights for players, teams, and analysts. The objective is to extract meaningful patterns and create an interactive dashboard for visualization. The dataset underwent preprocessing in Python using Pandas to handle missing values, normalize data, and create derived metrics like batting strike rates and bowling economies. The cleaned data was then imported into Power BI to design an interactive dashboard. The dashboard includes features such as: Player Performance Analysis: Batting strike rates and bowling economies. Match Insights: Key statistics like team scores, run rates, and partnerships. Dynamic Filtering: Year, team, and player-specific insights.

This project leverages advanced data manipulation and visualization techniques to provide a comprehensive view of T20 cricket performance.

Sales Dashboard for 'The Bike Haven'

Tools:

Tableau, SQL


This project focuses on creating an interactive and visually informative dashboard for The Bike Haven, a bike shop in San Francisco, to analyze its sales performance and track its business growth. The objective is to provide management with actionable insights into product, customer, and regional performance, and to compare actual sales with budgeted sales.

This dashboard empowers The Bike Haven with:
Better Decision-Making: Management can easily track sales performance and make data-driven decisions.
Enhanced Customer Insights: Identifies top customers, enabling personalized marketing and loyalty efforts.
Product and Regional Strategy: Helps focus on the highest-performing products and regions for targeted growth strategies.

IPL Auction Strategy Using SQL

Tools:

SQL


This project focuses on analyzing IPL player performance data to support a new franchise in developing a competitive auction strategy. The objective is to identify and rank players across different roles—batters, bowlers, all-rounders, and wicketkeepers—using SQL queries and data analysis techniques. The dataset was loaded into a SQL database for structured querying. SQL was used to filter, process, and analyze player performance metrics, such as strike rates, bowling economies, and boundary percentages. The insights derived were organized in Excel for visualization and presentation.

This project leverages structured data analysis and SQL to empower the franchise with actionable insights for building a competitive and balanced IPL team.

Sales Insights Dashboard

Tools:

Power BI


This project focuses on analyzing and visualizing sales performance data to provide actionable insights for businesses. The objective is to enhance decision-making by identifying key trends and patterns in sales data.

Key features of the dashboard include: Sales Overview: Visualizing total sales, profits, and revenue distribution across regions and products. Customer Insights: Identifying top customers based on sales and purchase frequency. Product Analysis: Highlighting best-performing products and categories. Time-Based Trends: Yearly, monthly, and quarterly sales analysis to track growth and seasonality. The dashboard leverages interactive filters and drill-down capabilities, enabling stakeholders to explore specific aspects of the data in detail.
