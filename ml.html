Rainfall Prediction Using Machine Learning

Tools:

Scikit-Learn, Jupyter Notebook, Machine Learning Algorithms


This project aims to predict rainfall using historical weather data. The goal is to assist in planning and decision-making in agriculture and disaster management.

The dataset underwent basic Exploratory Data Analysis (EDA) to uncover key trends and patterns. A Random Forest Classifier from the Scikit-Learn library was used to develop a predictive model, achieving an accuracy of 82.3%

After conducting basic EDA, a Random Forest Classifier model from the Scikit-Learn library was built to predict rainfall, leveraging key weather features to achieve an accuracy of 82.3%.
